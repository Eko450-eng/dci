Using HTML

What is HTML?

HTML stands for Hypertext markup language, it is used for structuring a website and getting the basics for it done such as textstyling and implement text.
Using HTML only you will be easily able to make a website that I would compare to a bannerpaper
HTML is using elements that are representated by "tags" here are the most common and basic ones.

Tags -The most important tool

before we start you need to understand how tags are built.
Most tags have an opening and a closing tag but not every tag has a closing tag, if you use programms like visual studio code those will be implemented on their own, but if not you will need to know which ones do have closing tags and which dont. An example for a tag with a closing tag would be an anchor it looks like this. <a href="www.yourlink.de">Click here</a> As you can see the closing tag is like an opening tag but start with / Tags that don't need a closing tag will in this tutorial be marked with *

Head

Lets start at the most confusing part. The Head

The HTML <head> element has nothing to do with HTML headings (h1, h2,...)! The <head> element is a container for Metadata (meta tags) and other informations about the HTML document. Metadata are not displayed in the browser window.

Inside the <head> ... </head> you can for example place the following elements:

  • <title> Defines the page title (important for SEO)
  • <style> Style information for your webpage
  • <link> Used to link to external stylesheets (see CSS), favicon, Google fonts etc.
  • <meta> Defines meta data like character set, description, keywords, author and more
  • <script> Defines client side JavaScript for your page

Headers

Headers work using the <h(number from 1-6)> tags with increasing number the importancy of the header decreases, headers need a closing tag

You should always only have 1 h1

Comments

Comments are made with <!-- YOUR COMMENT--> they are used to put text into your code without the text interfering with it or being rendered on the page its just for the developer a sort of note

Tables

Tables are just like the name implies a way of displaying tables on your website

These are the Table elements

  • <table> Defines a table
  • <tr> Defines a table row
  • <td> Data container. Can contain all sorts of HTML elements
  • <th> Defines a table heading
  • <caption> Defines a table caption
  • <colgroup> Specifies a group of columns in a table for formatting
  • <col> Specifies column properties for each column within a element

A simple table could therefore look like this

<table>

<tr>

<th>First Name</th>

<th>last Name</th>

<th>Age</th>

</tr>

<tr>

<td>Eve</td>

<td>Smith</td>

<td>46</td>

</tr>

</table>

The output looks like this

First Name Last Name Age
Eve Smith 46

Lists

Lists are very usefull

We have 2 basic types of lists, ordered lists and unordered lists

<ol> defines and ordered list

<ul> defines and unordered list

and <li> defines a list item

Ordered Lists

<ol>

<li>First item</li>

<li>Second item</li>

<li>Third item</li>

</ol>

The output looks like this:

  1. First Item
  2. Second Item
  3. Third Item

Unordered Lists

<ul>

<li>Home</li>

<li>Products</li>

<li>Contact</li>

</ul>

The output looks like this:

  • First Item
  • Second Item
  • Third Item
